Examples

Basic Drag/Swap/Delete

<div id='twoListDiv'> </div>
const dl1 = new DraggableListGenerator() dl1.createList({parent: document.querySelector('#twoListDiv'), listName: 'CSC309 Section 1 Student List', entries: ['Mark', 'John', 'Tony', 'Tom', 'Zack', 'James'], background_color: '#347f94'}) dl1.enableDrag() const dl2 = new DraggableListGenerator() dl2.createList({parent: document.querySelector('#twoListDiv'), listName: 'CSC309 Section 2 Student List', entries: ['Viki', 'Raymond', 'John', 'Ben', 'Alex', 'Sam'], background_color: '#c8d8e4'}) dl2.enableDrag()
1. Create draggable list by specifying the parent element, listName, cellWidth/Height, entries, and background_color. 2. However, these parameters are all optional, with default values associated with all of them. 3. After a list has been created by the DraggableListGenerator, you can enable drag editing by calling the enableDrag() function. 4. You can swap list entires by dragging and dropping them on top of other draggable lists. 5. You can move a list entry to another draggable list by dropping it onto the add entry area located at the bottom of the draggable list. 6. You can also delete a list entry by dropping in onto the garbage icon.

Merge/Unmerge of Entries' Content

<div id='mergeListDiv'> </div>
const dl3 = new DraggableListGenerator() dl3.createList({parent: document.querySelector('#mergeListDiv'), listName: 'CSC309 Study Partner Assignment', entries: ['Viki', 'Raymond', 'John', 'Ben', 'Alex', 'Sam'], background_color: '#c8d8e4'}) dl3.enableDrag()
1. You merge the content of list entries together by dragging and waiting. 2. Start by dragging one of the entries (moved entry) that you want to merge onto the second entry (target entry) that you want to merge. 3. After 2.5 seconds, a prompt will show up indicating that merging could happen. 4. You can then release the mouse button to have the entires merged together, the merged entries are now considered as one entry and can be moved, swapped, or merged the same way. 5. When hovering over individual entry content, is opacity will change, indicating that they can be unmerged. 6. To unmerge previously merged content, click and hold the mouse button down on the target content that you want to unmerge, make sure not to move your mouse. 7. After 2.5 seconds, the target content will disappear and form a new list entry which you can drop before any list entries or toss it into the garbage to delete.

Entries Style Modification

<div id='styleListDiv'> </div>
const dl4 = new DraggableListGenerator() dl4.createList({parent: document.querySelector('#styleListDiv'), listName: 'Different Style List', entries: ['Amaranth Pink', 'Ash Gray', 'Beau Blue', 'Bittersweet']}) dl4.changeStyle("font-weight: bold;") dl4.changeStyle("backgroundColor: #F19CBB; color: #1F75FE", 0) dl4.changeStyle("backgroundColor: #B2BEB5", 1) dl4.changeStyle("backgroundColor: #BCD4E6", 2) dl4.changeStyle("backgroundColor: #FE6F5E", 3)
1. You can change style of entries by calling function changeStyle(style, idx). 2. Styles are specified in the form of "styleName: styleValue" 3. You can change multiple styles at once by seperating them with ";" 4. You can specify which entry to change by specifying the index parameter. 5. The default value of -1 will apply the style change to all list entires.
